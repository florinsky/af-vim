# Plugins Keys

## Panels:
* C-t - toggle TAGs panel
* C-e - toggle file explorer

## Popup:
* C-p - search files (by name)
* C-f - find in current file
* C-g - find in all opened files
* C-b - buffer list

## Auto-complete (php):
* C-space

## Search under cursor:
* CR - toggle highlight the word under cusor

## Usefull Commands:
* :noh - remove highlighted search word

## Fugitive (commands):
* :Gstatus

## Snippets (tab key):
* ec - echo "string";
